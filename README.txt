# Step Calculator 
## Data Prep
This project cleans a TRC file for post-production and analysis.
Main Tasks
- Remove Superfilous header rows
- Fill in NAN/Blanks in motion data
- Calculate New Metrics for Step length and Width
- (TODO) Body Stability
- Selected Ranges of Frames
- Selected Single or Multiple Markers for analysis.

## Usage

#### python run_for_all_files.py
